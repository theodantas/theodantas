Sobre mim

Me chamo Théo Dantas, sou estudante do Bacharelado em Ciência & Tecnologia pela UFABC, com direcionamento para a área de Ciência da Computação e foco em Dados. Tenho base sólida em lógica de programação, SQL e Python, desenvolvendo projetos voltados à análise e estruturação de dados para geração de soluções eficientes. Busco me aprofundar na área de Dados, onde pretendo evoluir tecnicamente e gerar impacto através de soluções orientadas por informação.

Conhecimentos

HTML5

HTML5 é uma linguagem de marcação de hipertexto, responsável por estruturar o conteúdo de uma página web. Ele organiza textos, imagens, links e seções, definindo a base e a hierarquia das informações exibidas no navegador.

CSS3

CSS3 é uma linguagem de estilo usada para a parte visual. Ele controla cores, fontes, espaçamentos, layouts e animações, permitindo criar um design moderno e responsivo.

JavaScript

JavaScript é uma linguagem de programação que adiciona interatividade às páginas web, sendo possível criar efeitos, validações, atualizações dinâmicas e tornar a experiência do usuário mais funcional.

MySQL

MySQL é um sistema de gerenciamento de banco de dados relacional usado para armazenar, organizar e consultar informações. Ele permite trabalhar com dados de forma estruturada usando SQL.

Python

Python é uma linguagem de programação simples e fácil de aprender, usada para criar sites, sistemas, automações, jogos e aplicações de inteligência artificial, sendo muito popular por sua praticidade e grande quantidade de bibliotecas disponíveis.
